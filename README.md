## Overview

A Tkinter-based PDF utility offering viewing, text/image scraping, editing,
search, OCR and more.

## Requirements

```bash
pip install PyMuPDF Pillow
# optional for OCR
pip install pytesseract
```

## Running

```bash
python PDFScraperPro.py
```
